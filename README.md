# 2. Obsidian Git User Guide

## 2.1. Windows 10 on PC

1. Install Git
2. `git clone from https://github.com/NeonHo/CSMLNotebook.git`
3. Open the repository as the obsidian vault.
4. Install **Obsidian Git** plugins.
5. Enable the plugins.
