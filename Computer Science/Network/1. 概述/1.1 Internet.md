# 2. 计算机网络
联网的计算机构成的系统
- 节点
	- 主机节点
		- 是数据源；
		- 是数据目标。
	- 数据交换节点
		- 中转节点：转发数据；
		- 既不是源也不是目标；
		- 实现从源到目标的数据传输。
- 链路
	- 将节点连在一起。
	- 接入网链路：
		- 主机节点——交换节点
	- 骨干链路：
		- 交换节点——交换节点
- 协议
	- 支撑网络工作的标准；
	- 厂商根据标准生产设备；
	- 节点之间因此可互操作。
